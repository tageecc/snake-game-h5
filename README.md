# 🐍 贪吃蛇 H5 游戏

经典的贪吃蛇游戏，支持 PC 和移动端。

## 🎮 在线游玩

- **GitHub Pages**: https://tageecc.github.io/snake-game-h5/
- **Vercel**: 部署中...

## ✨ 功能特性

- 🎨 Canvas 渲染，60fps 流畅运行
- 🎮 键盘控制（方向键/WASD）
- 👆 触摸控制（滑动 + 虚拟按钮）
- 📱 响应式设计（PC/移动端/微信）
- 🏆 最高分记录（localStorage）
- ⚡ 难度递增（每 50 分加速）

## 🛠️ 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6+)
- CSS3 响应式布局

## 🚀 本地开发

```bash
# 克隆仓库
git clone https://github.com/tageecc/snake-game-h5.git

# 直接用浏览器打开 index.html
# 或使用本地服务器
npx serve .
```
